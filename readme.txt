LPTK - Light Pascal ToolKit

What is LPTK?

  LPTK is a native Pascal language written library that provides 
  several functionality for creating Windows and/or Linux/FreeBSD 
  GUI applications. 

  LPTK uses the lowest level API-s, it has own look, it depends on
  only few other smaller libraries (Xlib, Xft on Linux).
